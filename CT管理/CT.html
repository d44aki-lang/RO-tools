🚀 初期設定の方法

本ツールを使い始めるためのキャラクター登録手順です。

1. RO公式からキャラ情報を取得

Ragnarok Online公式の「キャラクター情報」ページを開き、キャラクター一覧が見える状態でスクリーンショットを撮影します。

2. AI（Gemini等）でJSON変換

以下のプロンプトを添えて、スクリーンショットをAIに投稿してください：

この画像に写っているキャラクター名、BaseLv、職業をすべて抽出し、以下のJSON形式で出力してください。
余計な解説は不要です。コードブロックのみ出力してください。

[
  { "name": "キャラ名1", "lv": 250, "job": "職業名", "order": 0 },
  { "name": "キャラ名2", "lv": 240, "job": "職業名", "order": 1 }
]

3. ツールに貼り付け

出力されたJSONをコピーし、本ツールの「👤 キャラ設定」タブにある「JSON 一括インポート」欄に貼り付けて「一括登録」ボタンを押せば完了です。

🗓️ MD設定について

「🗓️ MD設定」タブでは、管理したいMDの追加や編集が可能です。

  • MD名 / MVP数: MD完了時に加算される討伐数を設定できます。
  • CT (クールタイム): 毎日05:00、3日後05:00、火曜12:00のいずれかを選択すると、チェックが自動的に外れるようになります。
  • BaseLv / 職業名: キャラ設定タブで入力・編集が可能です。
  • 表示設定: キャラ設定タブで進捗画面へのLv・職の表示有無を切り替えられます。
  • 並び替え: ドラッグ＆ドロップで表示順を自由に変更できます。
  • 公式MDリスト適用: 主要なMDセットをワンクリックで登録できます。

🎮 通常利用方法

📋 MD進捗の管理

左クリック: ✔ (完了) ↔ × (未完了)
右クリック: － (対象外) の切替
🔄 縦横入れ替え: 表の行列を入れ替えて表示します。
完了状態(✔)にすると、設定されたMVP数が自動的に「討伐数」に加算されます。

👑 MVPカードの記録

カードが排出されたら「記録してリセット」を押すと、現在の討伐数を引き継いで履歴に保存されます。
「売却額」などは履歴一覧から直接編集可能です。
