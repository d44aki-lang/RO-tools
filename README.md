# RO-tools

Ragnarok Online (RO) のプレイを効率化するための便利ツール集です。
主にメモリアルダンジョン（MD）の周回管理の記録をサポートします。

## 収録ツール

### 1. RO MD管理ツール (`CT管理/CT.html`)

ブラウザ上で動作する、高機能なMD進捗管理およびMVPカード獲得記録ツールです。
IndexedDBを使用しているため、データはブラウザ内に保存され、ページを閉じても維持されます。

#### 🚀 主な機能

-   **MD進捗の一括管理**:
    -   全キャラクターのMD完了状況をグリッド形式で可視化。
    -   左クリックで「完了（✔）」と「未完了（×）」を切り替え。
    -   右クリックで「対象外（－）」に設定可能。
-   **自動クールタイム（CT）リセット**:
    -   ROの仕様に基づいた自動リセット機能を搭載。
    -   リセットタイミング：毎日 05:00 / 3日後 05:00 / 火曜 12:00
-   **MVPカード獲得履歴**:
    -   カードの排出日、カード名、討伐数、売却額を記録。
    -   年ごとの集計や、全期間の累計サマリー（総枚数、総討伐数、総Zeny）を自動算出。
    -   1枚あたりの平均討伐数などの統計も表示。
-   **高度なカスタマイズ**:
    -   **キャラクター登録**: AI（Gemini等）と連携し、公式サイトのスクリーンショットからキャラクター名の一括インポートが可能。
    -   **MD設定**: 管理するMDの追加・削除、表示順のドラッグ＆ドロップによる変更が可能。
    -   **テーマ切り替え**: 複数のカラーテーマ（Deep, Ocean, Forest, Blood, Snow, Ivory）から選択可能。
-   **データ管理**:
    -   JSON形式でのエクスポート/インポートに対応しており、バックアップや環境移行が容易です。

#### 🛠 使用技術

-   **Frontend**: HTML5, CSS3 (Vanilla CSS), JavaScript (ES6+)
-   **Storage**: IndexedDB (Local persistence)
-   **Design**: ダークモードを基調としたモダンなUI、レスポンシブデザイン

#### 📥 初めて使う方へ（導入手順）

GitHubに慣れていない方でも、以下の手順で簡単にツールを使い始めることができます。

1.  **ファイルをダウンロードする**:
    -   このリポジトリの [CT管理/CT.html](CT管理/CT.html) をクリックして開きます。
    -   画面右上にある **[Download raw file]** ボタン（下矢印のアイコン）をクリックして、PCに保存します。
2.  **ツールを起動する**:
    -   保存した `CT.html` をダブルクリックしてブラウザ（ChromeやEdgeなど）で開きます。
3.  **ブックマークに登録する**:
    -   ブラウザで開いたら、`Ctrl + D` キーを押してお気に入りに追加してください。
    -   次からは、そのブックマークをクリックするだけでいつでもツールを使えます。

---

#### 📖 使い方 (詳細)

1.  **初期設定**:
    -   「👤 キャラ設定」タブでキャラクターを登録します。
    -   「🗓️ MD設定」タブで「公式MDリストを適用」ボタンを押すと、主要なMDが自動登録されます。
3.  **日々の管理**:
    -   「📋 MD進捗」タブで、周回が終わったMDをクリックしてチェックを入れます。
    -   MD完了時に、設定された「MVP数」が現在の討伐数に自動加算されます。
4.  **記録の保存**:
    -   「👑 MVPｃ獲得」タブで「記録してリセット」を押すと、現在の討伐数を引き継いでカード獲得履歴に保存されます。
